# 模块 3 · 推理与思维链进阶

> **难度**：⭐⭐⭐ 中级
> **建议周次**：第 4–5 周（约 12 小时）
> **前置**：模块 1、2
> **产出能力**：让模型解决复杂推理问题

---

## 🎯 本模块目标

从"让模型回答"进化到"让模型**思考**"。

掌握 4 种进阶推理技法，应对以下痛点：
- 模型直接答错数学题
- 复杂判断只给结论不给理由
- 同一问题多次回答不一致

---

## 💼 应用场景

| 场景 | 推荐技法 |
|---|---|
| 数据分析逻辑推导 | CoT + Self-Consistency |
| 项目风险识别 | Tree of Thoughts |
| 股票 / 基金多因子分析 | Step-back + CoT |
| 代码 bug 诊断 | ReAct |
| 客户方案评估 | ToT + Rubric |

## 📚 核心知识点

### 1. Self-Consistency（自洽性投票）

**原理**：同一问题让模型用较高温度（temperature）独立回答 N 次，取多数答案。

**何时用**：答案空间较窄（数字、分类、选择题）。
**代价**：成本 ×N，延迟 ×N。

```
runs = 5 次
结果：A, A, B, A, C → 最终答案 A（3 票）
```

### 2. Tree of Thoughts (ToT)

**原理**：让模型列出多个推理路径，每条路径自评分，再走最优路径继续展开。

**适用**：规划、多步决策、方案比选。

**简化版 Prompt**：

```
针对这个问题，请：
1. 列出 3 种可能的分析路径
2. 对每条路径的可行性打分（1-10）
3. 选择得分最高的路径，给出最终答案
```

### 3. Step-back Prompting（退一步思考）

**原理**：先让模型抽象出"这类问题的通用原理"，再套用到具体问题。

**示例**：
- 原问题：这只股票近期下跌 15%，要不要止损？
- Step-back：先回答"止损决策一般考虑哪些因素？"
- 再问：结合这些因素，分析这只股票的具体情况。

**效果**：避免直觉式回答，调动模型的"知识结构"。

### 4. ReAct（Reasoning + Acting）

**原理**：交替输出 `Thought`（思考）和 `Action`（动作，通常是调用工具），是 Agent 的雏形。

```
Thought: 我需要先知道当前股价
Action: get_stock_price(symbol="AAPL")
Observation: $185.32
Thought: 再对比 5 日均线...
Action: get_ma(symbol="AAPL", days=5)
...
Final Answer: 建议...
```

模块 6 会详细展开。

## 🛠 学习步骤

### Step 1：CoT 升级版实操（2 小时）

把模块 2 的 Prompt 加一段 `<thinking>` 标签，强制模型先在里面列推理步骤，再给 `<answer>`。

```xml
<thinking>
...（模型在这里写推理）
</thinking>

<answer>
...（最终答案）
</answer>
```

### Step 2：Self-Consistency 实验（3 小时）

- 挑 10 道数学或逻辑题
- 分别用单次回答 vs 5 次投票
- 计算准确率提升

### Step 3：ToT 在投资分析上的应用（3 小时）

选一只股票，让模型：
1. 列出 3 种分析视角（基本面 / 技术面 / 宏观面）
2. 每个视角独立输出结论
3. 汇总并标注一致性和分歧点

### Step 4：Step-back 在销售场景的应用（2 小时）

遇到"客户说太贵"这种具体问题，先让模型抽象出"应对价格异议的通用框架"，再套用。

## 💻 示例：股票多路径分析 (ToT + Step-back)

```xml
<role>
你是一位资深基金经理，擅长多维度分析股票。
</role>

<step_back>
在开始分析具体股票前，先用 3-5 条列出「评估一只股票是否值得持有」的通用框架。
</step_back>

<task>
针对股票 {{symbol}}，沿以下 3 条路径独立分析：
1. 基本面（财报、估值、行业地位）
2. 技术面（趋势、成交量、关键位）
3. 宏观面（行业周期、政策、利率环境）
</task>

<synthesis_instruction>
最后：
- 列出三条路径的**共识**和**分歧**
- 给出一个综合判断（加仓 / 持有 / 减仓 / 观望），并说明理由
- ⚠️ 声明：本分析为学习研究用途，不构成投资建议
</synthesis_instruction>

<data>
{{输入的财务与行情数据}}
</data>
```

## 🏗 实战项目：多视角投资分析助手

**要求**：
- 输入：股票代码 + 最近 20 个交易日 K 线 + 最新财报摘要
- 处理：Step-back → ToT 三路径分析 → 综合
- 输出：结构化分析报告（Markdown）

**验收**：
- [ ] 3 条路径逻辑互相独立
- [ ] 明确标出一致性 / 分歧
- [ ] 不做具体买卖点推荐（合规）
- [ ] 同一数据跑 3 次，核心结论稳定（±10% 范围内）
